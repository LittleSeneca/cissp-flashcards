# SOC Reporting

This section covers System and Organization Controls (SOC) reports, which are critical for third-party risk management and security assessment.

## Key SOC Report Types
- **SOC 1**: Focused on **Financial Reporting** (ICFR). Based on SSAE 18.
- **SOC 2**: Focused on **Security, Availability, Processing Integrity, Confidentiality, and Privacy** (Trust Services Criteria). Detailed report for restricted use.
- **SOC 3**: A **publicly available** summary of the SOC 2 report (no detailed test results).
- **Type I**: A point-in-time assessment of **design** effectiveness.
- **Type II**: An assessment of **design and operating** effectiveness over a period (usually 6-12 months).

## Trust Services Criteria (TSC)
The five categories used in SOC 2 and SOC 3 reports:
1. **Security** (The "Common Criteria")
2. **Availability**
3. **Processing Integrity**
4. **Confidentiality**
5. **Privacy**

## SSAE 18
The Statement on Standards for Attestation Engagements no. 18, the standard used by auditors to perform SOC 1 examinations.
