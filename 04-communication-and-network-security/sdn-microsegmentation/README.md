# SDN & Microsegmentation

This section covers Software-Defined Networking (SDN) and Microsegmentation concepts relevant to Domain 4 of the CISSP.

## Key Topics
- **SDN Planes**: Control, Data (Forwarding), and Management planes.
- **SDN APIs**: Northbound (Application to Controller) and Southbound (Controller to Switches).
- **Microsegmentation**: Granular security policies applied at the workload level to prevent lateral movement.
- **VXLAN**: Virtual Extensible LAN, used for network virtualization and encapsulation.
- **SD-WAN**: Software-Defined Wide Area Network.

## CISSP Relevance
SDN decouples the network control logic from the underlying hardware, allowing for centralized management and increased agility. Microsegmentation is a key component of Zero Trust architectures, providing isolation within the data center.
